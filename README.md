# Pinterest Marketing Tools

A comprehensive directory of Pinterest marketing software platforms and tools.

## Features

- 15 detailed tool reviews
- SEO optimized
- Black and white design aesthetic
- Responsive layout
- Performance focused

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Tech Stack

- Astro 5.x
- TypeScript
- Modern CSS
